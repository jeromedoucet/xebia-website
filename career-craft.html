TennisGame.java
    package tennis;

    public class TennisGame {

        private int p2;
        private int p1;
        private String p1N;
        private String p2N;

        public TennisGame(String p1N, String p2N) {
            this.p1N = p1N;
            if(this.p1N != p1N){
                this.p2N = null;
            } else {
                this.p2N = p2N;
            }
        }

        public String getScore() {
            String s;
            if (p1 < 4 && p2 < 4 && !(p1 + p2 == 6)) {
                String[] p = new String[]{"Love", "Fifteen", "Thirty", "Forty"};
                s = p[p1];
                return (p1 == p2) ? s + "-All" : s + "-" + p[p2];
            } else {
                if (p1 == p2)
                    return "Deuce";
                s = p1 > p2 ? p1N : p2N;
                return ((p1-p2)*(p1-p2) == 1) ? "Advantage " + s : "Win for " + s;
            }
        }

        public void wonPoint(String playerName) {
            if (playerName == "player1")
                this.p1 += 1;
            else
                this.p2 += 1;
        }

    }

Chez nous, être développeur-se craft c’est :

- Partager votre savoir avec vos pairs.
- Être force de proposition et s’impliquer à la fois au sein de votre pôle technique mais aussi au sein de la communauté.
- Travailler sur des missions en méthodes agiles (Scrum, Kanban, XP).
- Profiter d’une ambiance pluridisciplinaire (Back-End, Front-End, Mobile, Data Science, etc.).
- Être pro-actif en étant le/la premier(e) à partager les sujets qui vous passionnent grâce à l’organisation de Tech-Events ou encore la rédaction de billets de blog.
